ReactAct - Start Servers Guide

Project path:
/Users/subrat/Desktop/ReactAct

1) Start Backend (Django)

Open Terminal 1 and run:

cd /Users/subrat/Desktop/ReactAct/backend
source /Users/subrat/miniconda3/etc/profile.d/conda.sh
conda activate agent
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver 8000

Backend URL:
http://127.0.0.1:8000

2) Start Frontend (React/Vite)

Open Terminal 2 and run:

cd /Users/subrat/Desktop/ReactAct/frontend
npm install
npm run dev

Frontend URL:
http://localhost:5173

3) Quick Check

- Login/Register on frontend.
- Dashboard and Builder should load after login.
- API health check:
  http://127.0.0.1:8000/api/health/
